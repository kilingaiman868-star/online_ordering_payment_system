#  Online Ordering & Payment System
A complete web-based online ordering and payment system built with PHP, MySQL, HTML, CSS, and JavaScript. Customers can view products, place orders, and select payment methods. All orders are stored directly in a MySQL database.

##  Features

-  Display products with images, descriptions, and prices
-  Customer order form with validation
-  Multiple payment methods (Airtel Money, M-Pesa, Tigo Pesa, Credit Card, Cash on Delivery)
-  Automatic database and table creation
-  Sample products inserted automatically
-  Orders saved directly to MySQL database
-  Responsive design for mobile and desktop
-  JavaScript form validation
-  Secure data sanitization

##  Technologies Used

| Technology | Purpose |
|------------|---------|
| PHP | Backend logic and database interaction |
| MySQL | Database storage |
| HTML5 | Page structure |
| CSS3 | Styling and responsive design |
| JavaScript | Client-side form validation |
| XAMPP | Local development server |

##  Requirements

- XAMPP (or any local server with PHP and MySQL)
- PHP 7.0+
- MySQL 5.6+
- Web browser (Chrome, Firefox, Edge, etc.)
